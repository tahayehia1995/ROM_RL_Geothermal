# ROM_RL_Geothermal

A project combining Reduced Order Modeling (ROM) and Reinforcement Learning (RL) for geothermal reservoir management.

## Project Structure

- **ROM_Refactored/**: Reduced Order Modeling components for geothermal reservoir simulation
- **RL_Refactored/**: Reinforcement Learning components using SAC (Soft Actor-Critic) algorithm

## Setup

See individual README files in each module for specific setup instructions.
